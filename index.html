Center List Viewer Center Administration
Description:
This function shows a form to input a certified username and password from the Chief Admin and displays
all the current Centers in HiperTec where the admin can choose to edit, add or print the list of centers.
Use:
In the first form use the following data:
user: Admin
password: Password
and click the 'Login' button.
ConnectionUtils.java Establishes a connection with HiperTec's database.
Utils.java Adds a header and footer.
CenterData.java Uses constructors and SQL to create a vector containing each center.
Centers.java Uses a form to input username and password to login into Center Administration.
CenterList.java Displays all the current centers in HiperTec and gives the option to edit, insert or print the center list.
EditCenter.java Uses a form to change information on a specific center chosen by the user that when inputed redirects you to NewUpdateCenter.java
NewUpdateCenter.java Uses the information provided by the user in the previous form and updates HiperTec's database.After updating the center it redirects you to CenterList.java where you can verify the database was updated after refreshing the page.
InsertCenter.java Uses a form where the user can input information to insert a new center, it then redirects you to InsertSuccessful.java.
CheckCenter.java Uses Ajax and Javascript to run the functions responsible for the centerID verification
InsertSuccessful.java Uses the information provided by the user in the previous form and updates HiperTec's database.After inserting the center it redirects you to CenterList.java where you can verify the database was updated after refreshing the page.
Export.js Uses JavaScript:

* When the 'Add New Center' is clicked it redirects to InsertCenter.java: Add_Center()

*When the 'Download Results - PDF' button is clicked it prints the screen and a download option is available: downloadPDFWithBrowserPrint()

*When the 'Edit Centers' button is clicked it calls createButtons_Centers() which adds a new column with 'Edit Center' buttons that redirect you to EditCenter.java when clicked:createButtons_Centers()-Edit_Center(num)

* When the 'Update/Insert' buttons are clicked a Pop-up will appear asking the user if they would like to confirm their change which then redirects the user to CenterList.java: Confirm_Update()

* checkId() and checkAppIdAjax(Id) verify if the centerID is already introduced in HiperTec's database to avoid data input errors
