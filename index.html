Application List Viewer: Application List
Description:
This function shows a form that lists all current applications to the company, coming from many candidates that want to apply for a job at HiperTec.
Use:
By scrolling down, you can see all current applications in the database, and you can go up by clicking on the 'Back to top' link. The 'Add application' button redirects to another servlet in order to introduce an application into the database.
applicationsSearch.js Uses JavaScript in order to make a 'quick search' by introducing all sorts of parameters, including appId, candidateId, center, section, HHRR and date.

Add an application: Add application
Description:
This function shows a form to enter the details of an application manually. This registers the information in the database and shows an html confirming the success of this operation. If one adds an application id that is already taken the 'AjaxLuconi' file will make sure you choose another one in order to move forward.
Use:
You can access this function by clicking on the 'Add application' button in the ApplicationList servlet. In the first input of the form, use the following 'taken' id: 65999192
Now, use the following one (which is not taken): 65999195
Then submit the form after filling in the other fields.
AjaxLuconi.js Uses Ajax and Javascript to run the functions responsible for the id verification.
ApplicationAddedSuccessfully.html Confirmation message of operation.
CheckUserAjax.java Makes sure that the id input being introduced is not taken already.
ApplicationUpdate.java Updates the information in the database.

Candidate Profile Viewer: View candidate (id=221836)

Description:
This function shows a form that shows the details of a specific candidate. For the sake of the example, I have provided an id in the url already.
Use:
First, you have to access this function by the CandidateList servlet and click on any 'View application' link. We can click the 'Continue application' and a table will pop up with three more functions. We can accept, reject, or even edit this candidate profile. We also have two links at the bottom which redirects to the 'all candidates' and 'all applications' servlets.
CandidateOptions.js Uses Javascript to add a table below the first one by clicking the 'Continue application' button, and adds three more buttons within the second row to perform more functions.

Accept a candidate: Accept candidate (id=221836)

Description:
This function shows a form to submit the details of an newly accepted candidate manually. This registers the information in the database and shows an html confirming the success of this operation.
Use:
You can access this function by the ViewCandidate servlet with an id, and click on the 'Continue application' button to access the 'accept' button. This accept button will redirect you to the function with the candidate information in the id attribute. By editing the information and clicking the button the information is sent to the database.
CandidateAddedToWorkers.html Confirmation message of operation.
AddCandidatetoWorkers.java Updates the information in the database.
